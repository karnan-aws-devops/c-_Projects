# ATM Management System (C++)

A console-based ATM application developed using C++ to simulate basic banking operations.

## Features
- Secure PIN authentication with limited attempts
- Cash withdrawal with input validation
- Balance inquiry
- PIN change functionality
- Menu-driven user interface

## Technologies Used
- C++
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Compile using a C++ compiler (g++)
3. Run the executable

This is a self-learning academic project.
